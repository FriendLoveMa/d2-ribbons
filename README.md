D2 Ribbons 是一套为开发者准备的开源社区绶带资源，你你可以下载图片到你的项目中使用或者直接使用仓库资源链接。

## Features

* 扁平化风格
* Github Gitlab Gitee 三套图标
* 六色主题
* 半透明 / 不透明样式
* 文字 / 图标样式
* @2x 资源
* 统一尺寸（150 x 150 / 300 x 300 @2x）
* 附带 Github 经典样式的绶带
